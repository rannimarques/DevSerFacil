GLOSSÁRIO

  • Git

    • É uma ferramenta de controle de versão, o seja, ele ajuda com que você não precise de inúmeras pastas para salvar o seu projeto a cada modificação.




    via GIPHY


  • GitHub

    • É a ferramenta mais famosa para controle de versão que permite uma integração com a comunidade, ou seja, ela é como uma rede social onde programadores colocam seus projetos, trabalham juntos, contribuem com o trabalho do outro e claro, dão uma olhadinha no que há de novo.

    via GIPHY

  • Repositório

    • É o local dentro do git onde guardamos os conteúdos que queremos salvar. O usuário pode ter quantos repositórios forem necessários para guardar seus trabalhos/projetos


    via GIPHY

  • Commit

    • É o ato de enviar e guardar, ou seja, enviar dados ou códigos para armazenamento em um banco de dados ou em um sistema de controle de versão.


    via GIPHY

  • Push

    • É usado para enviar conteúdo do repositório local para um repositório remoto

    via GIPHY

  • Pull

    • É usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais.

    via GIPHY
